PSU Pozyx

This is some of the software that Sticks and Stones Software's Gabriel Mukobi built while interning at Portland State University. Gabe developed, implemented, and tested Pozyx, a high accuracy localized positioning system, with applications as a learning tool for hands-on college-level physics courses, as an advanced position and motion data collection device for professional athletes, and as a solution for other high precision localized positioning applications.
